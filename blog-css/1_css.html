# 1. meta

  1. charset　字符集
    <meta charset="utf-8">
    1
  2. viewport 视口
    <meta name="viewport" content="width=device-width,initial-scale=1.0,maximum-scale=1.0,user-scalable=no">
    1
    • width=device-width -> 宽度等于设备宽度
    • initial-scale=1.0 -> 初始缩放为1
    • maximum-scale=1.0 -> 最大缩放为1
    • user-scalable=no -> 用户不能缩放

# 三. 元素分类

# 1. 按默认样式分

  1. 块级 block

    • 方块形状
    • 默认占据一整行，不会给其他元素留出空间
    • div/

  2. 行内 inline

    • 不一有规则形状
    • 与其他元素共享一行
    • p/span/em/strong/a

  3. inline-block

    • 对内表现block
    • 对外表现inline
    • select/
