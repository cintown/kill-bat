## 位运算

`位运算知识点`

### 运算规则

| 符号 | 描述 |      运算规则      |
| :--: | :--: | :----------------: |
|  &   |  与  |   都为 1，才为 1   |
|  \   |  或  |   都为 0，才为 0   |
|  ^   | 异或 | 相同为 0，不同为 1 |
|  ~   | 取反 |   0 变 1,1 变 0    |
|  -   | 补数 |       取反+1       |
|  <<  | 左移 | 高位丢弃，低位补 0 |
| _>>_ | 右移 | 低位丢弃，高位补 0 |

`或的符号为 ‘|’`
