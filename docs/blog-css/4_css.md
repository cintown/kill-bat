## 二. 元素显示
1. display 元素显示类型
	* block/inline/inline-block
2. position 元素位置
	* static
	* relative : 1. 相对于元素本身偏移 2. 不会改变本身占据的空间
	* absolute : 1. 对文档流不造成影响 2. 相对与上级最近的relative/absolute定位
	* fixed : 1. 对文档流不造成影响 2. 相对于可视屏幕
	* relative/absolute/fixed 可设置 `z-index`,数值越高越在上层
