# 1607年
## 2月18日
### 明万历广宁门事件
　　1607年2月（明万历三十五年正月），泰兴县知县龙镗赴京参加“大计”（外官进京接受皇帝考核），因考迹不佳，被贬了官，心中郁闷不乐，带病离京。<br>　　当他走到广宁门，遇到税监赵禄、王泰、张宪、邢相等人正在卡门收税，见龙镗到来，误认为这个贬官大有油水可捞，于是便横加勒索。谁知龙镗的行囊空匮，满足不了税监们的贪心，开始仅对他污辱谩骂，接着拳打脚踢，还动用凶器。龙镗被打得篷头散发，口喷鲜血，昏死过去。事后群众发难，联名写了奏文，由兵部委请皇帝，要求依法严办这伙横行不法的税监。明神宗迫于公愤，拖了七八天，才把肇事者收监下狱。<br>　　经吏部查明：龙镗致命的一击出自赵禄之手。经过科道、法司合议，于2月18日（正月二十二日），将赵标处决，邢相等3人流放边卫充军。
## 4月26日
### 英国在北美建立弗吉尼亚殖民地
　　1584年，英国女王伊丽莎白一世为占有美洲未知土地，令其宠臣罗利爵士派遣探险队考察北美东部沿海一带。后罗利将北卡罗来纳以北之地，命名为弗吉尼亚，意即宜于移民垦殖和定居的处女地。<br>　　1607年，英国有3艘船只行驶到北美，殖民者史密斯等在弗吉尼亚亨利角登陆，并定居下来。<br>　　1607年4月26日，英国在北美建立了第一个殖民地——弗吉尼亚。<br>　　在以后的100多年间，英国在北美大西洋沿岸一共建立了13个殖民地。
