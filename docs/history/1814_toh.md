# 1814年
## 4月17日
### 纂辑《全唐文》告成
　　大学士董浩等纂辑《全唐文》成书。以《唐文》为蓝本，并采辑《永乐大典》、《文苑英华》、《唐文粹》等书而成。全书一千卷，共收作家三千余人，文一万八千四百余篇。
## 5月4日
### 拿破伦被放逐厄尔巴岛
　　拿破仑于1804年建立法兰西帝国，就任皇帝。<br>　　他对内巩固资产阶级革命成果，对外不断发动战争。<br>　　1812年大举进攻俄国遭到惨败。<br>　　1813年奥、英、俄等组织反法联盟进攻法国。10月在莱比锡决战中，拿破仑战败。<br>　　1814年3月反法联盟军兵临巴黎城下，拿破仑被迫于4月6日签署退让诏书，5月4日被放逐至地中海中的厄尔巴岛。<br>　　联盟军根据枫丹白露条约，规定厄尔巴岛作为拿破仑拥有三权的公爵领地，每年由法国复辟的波旁王朝提供 20O万法郎年金和 400名卫士。
