# 1815年
## 6月18日
### 滑铁卢战役爆发
　　滑铁卢战役是1815年拿破仑一世同第七次反法同盟联军进行的最后一次会战，也是拿破仑压到彻底失败的一次战役。<br>　　这次战役在拿破仑的“百日王朝”期间发生。第七次反法同盟一奥俄英普等国约集达百万人的军事力量，拿破仑亦动员法国的金部兵力（约50万）迎击联盟军。<br>　　6月18日在布鲁塞尔以南约20公里的滑铁卢村，拿破仑率军7．2万人同英国威灵顿率领的联盟军6．8万人和普鲁土军约4．5万人展开了决定性的会战。<br>　　最终法军溃败，拿破仑军伤亡2．5万人，被俘9000人。6月22拿破仑被迫第二次退位，法兰西第一帝国告终。
## 12月30日
### 姚鼐逝世
　　姚鼐(1732-1815)，字姬传，安徽桐城人。乾隆二十八年进士，曾为《四库全书》纂修官。鼐精于文，为文高简深古，尝论学问之事有三端，即义理、考据、文章，为桐城派主要作家。著《借抱轩全集》，编《古文辞类纂》等。
