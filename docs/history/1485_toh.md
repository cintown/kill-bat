# 1485年
## 8月22日
### 英国玫瑰战争站束
　　玫瑰战争（1455－1485）是英国历史上两大王室间的封建混战。兰开斯特家族的族徽为红玫瑰，约克家族的族徽为白玫瑰，这场两大王族之间的混战称玫瑰战争。<br>　　15世纪，英国的封建领地经济日趋衰落，大贵族便利用在百年战争期间建立的私人武装，谋取权力与财富。<br>　　1455年5月，约克公爵理查起兵反对兰开斯特家族的亨利大世，战争爆发。<br>　　1460年理查战死。<br>　　1461年3月，理查之子爱德华占领伦敦，废黜亨利六世，取得王位，称爱德华四世，建立约克王朝。<br>　　1483年4月，爱德华四世卒，其子爱德华五世即位，由其弟理查摄政。<br>　　6月，理查篡位，称理查三世。约克家族中拥护爱德华四世后裔的贵族和兰开斯特家族的余党联合，与理查抗衡。<br>　　1485年8月22日，兰开斯特家族的远亲亨利·都铎击败理查，自称为王，为亨利七世，建立都铎王朝。<br>　　玫瑰战争结束。
