# 884年
## 8月18日
### 黄巢失败自尽
　　黄巢（？－884），曹州冤句人，私盐贩出身。<br>　　875年率众响应王仙艺起义。后王仙艺战死，黄巢被义军推为领袖，自称“冲天太保均平大将军”。<br>　　878年黄巢采用流动作战战术，避实击虚，向江南地区进军，攻浙西，占福州，破广州，义军队伍发展到60万人。后回师北上，越湖南，跨淮河。<br>　　880年占洛阳，唐僖宗逃往四川。<br>　　881年起义军进长安，建立政权，国号大齐。废僖宗集中兵力，围攻长安，起义军将领朱温叛变投敌。<br>　　883年，黄巢被迫撤出长安，最后于公元884年8月18日（唐中和四年七月二十四日）在山东泰山粮虎谷战败自尽。
